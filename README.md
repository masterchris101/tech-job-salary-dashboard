# Tech Job Salary Dashboard

An interactive dashboard built with Python and Streamlit to explore tech job salaries by role, experience level, and remote work.

## Features
- Filter by job title, experience level, and remote ratio
- Key metrics (total jobs, average salary, max salary)
- Salary comparisons across roles
- Salary distributions by experience and remote work
- Dynamic insights based on selected filters

## Tech Stack
- Python
- Pandas
- Streamlit
- Plotly

## Dataset
Public tech salary dataset containing global job roles, compensation, experience levels, and remote ratios.

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
